Centre outils technicien

Cette page regroupe les commandes pour créer une clé USB RescueGrid, lancer le menu WinPE, préparer le PXE et déployer sur Synology.

1. Créer clé USB

Prépare une clé technicien avec scripts RescueGrid, menu WinPE, dossiers de rapports et outils externes.

powershell -ExecutionPolicy Bypass -File agent\windows\Create-RescueGridUSB.ps1 -TargetDrive E: -DashboardUrl "http://192.168.1.10:8000" -IncludeProject

2. Menu WinPE

Lance le menu atelier depuis la clé ou depuis WinPE.

powershell -ExecutionPolicy Bypass -File agent\windows\Start-RescueGrid.ps1

3. PXE Rescue Server

Préparation du serveur de boot réseau si le module PXE est disponible.

powershell -ExecutionPolicy Bypass -File pxe\Setup-PXERescueServer.ps1
